Описание:

В репозитории представлены проекты демокурса Аналитик данных (Karpov.Courses).

Основные инструменты и навыки, полученные при обучении:

- Языки: Python
- Анализ данных: библиотеки Pandas
- Визуализация: Matplotlib, Seaborn

Проекты:

1. Бронирование отелей
О чём проект: Анализ данных о бронировании отелей
Инструменты и навыки: Python, Pandas, предобработка данных, анализ данных

2. Программа лояльности
О чём проект: Анализ данных о покупках для программы лояльности
Инструменты и навыки: Python, Pandas, Matplotlib, Seaborn, предобработка данных, анализ данных, визуализация данных
